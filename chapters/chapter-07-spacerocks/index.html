Space Rocks

Ein Asteroids-Klon und die Zusammenfassung des ganzen Kapitels: Schub in Blickrichtung, Bildschirm-Wrapping, Geschosse, Schild, Leben, Level mit steigender Geschwindigkeit. Original: A3GPU207_SpaceRocks.

Schub statt direkter Bewegung

Der Unterschied zu Chapter 07a "Fahrendes Auto": Das Schiff bewegt sich nicht direkt in Blickrichtung, sondern ↑ erhöht eine Geschwindigkeit (shipMoveX/Y), die dann über mehrere Frames wirkt — echtes Trägheits-Gefühl statt "Stop-and-Go". Genau wie im Original bleibt die Rotation davon unabhängig steuerbar.

ActionScript 3 
if (upArrow) {
  shipMoveX += Math.cos(
    Math.PI*ship.rotation/180
  )*thrustPower;
  shipMoveY += Math.sin(
    Math.PI*ship.rotation/180
  )*thrustPower;
}
ship.x += shipMoveX;
ship.y += shipMoveY;
JavaScript 
if (upArrow) {
  const rad = deg2rad(rotation);
  shipMoveX +=
    Math.cos(rad)*thrust*dt;
  shipMoveY +=
    Math.sin(rad)*thrust*dt;
}
ship.x += shipMoveX * dt;
ship.y += shipMoveY * dt;

Bildschirm-Wrapping

Verlässt ein Objekt eine Kante, taucht es auf der gegenüberliegenden Seite wieder auf — Schiff, Felsen und (im Original) auch die Geschosse. Eine kleine wrap()-Hilfsfunktion ersetzt die vier einzelnen Grenzfall-Prüfungen aus dem Original für Schiff und Felsen gemeinsam.

Felsen teilen sich beim Treffer

Großer Felsen → zwei mittlere, mittlerer → zwei kleine, kleiner verschwindet ganz. Dieselbe rekursive Aufteilung wie im Original, nur dass die Felsenform hier zufällig gezackt statt aus einem Bibliothekssymbol gezeichnet wird.

←/→ drehen, ↑ Schub, Leertaste feuert, Z aktiviert ein Schild (3 Sekunden Schutz, begrenzte Anzahl pro Leben).

3 Schiffe, Felsen die sich teilen. Wie weit kommst du?
