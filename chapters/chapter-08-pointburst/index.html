Point Burst

Eine kleine, wiederverwendbare Komponente: Punktezahl an einer Position einblenden, während sie größer wird und ausblendet. Original: A3GPU208_PointBurst — wird danach in Collapsing Blocks und Match Three wiederverwendet.

Eine sich selbst verwaltende Komponente

Das Interessante an PointBurst.as ist, dass es sich nach der Animation selbst wieder entfernt — der Aufrufer muss sich um nichts weiter kümmern. Im Original per Timer mit fester Schrittzahl, hier als Objekt mit done-Flag, das die aufrufende Schleife aus einer Liste entfernt.

ActionScript 3 
animTimer = new Timer(
  animStepTime, animSteps);
animTimer.addEventListener(
  TimerEvent.TIMER,
  rescaleBurst);
animTimer.addEventListener(
  TimerEvent.TIMER_COMPLETE,
  removeBurst);
animTimer.start();

function rescaleBurst(e) {
  var p = e.target.currentCount
    /animSteps;
  scaleX = scaleY =
    (1-p)*startScale
    + p*endScale;
  alpha = 1-p;
}
JavaScript 
update(now) {
  const p = Math.min(
    (now-this.start)/animTime,
    1);
  this.scale =
    (1-p)*startScale
    + p*endScale;
  this.alpha = 1-p;
  if (p >= 1) this.done = true;
}

// im Game-Loop:
bursts.forEach(b => b.update(now));
bursts = bursts.filter(
  b => !b.done);

Aufruf bleibt denkbar einfach

Genau wie im Original reicht ein Konstruktoraufruf mit Punktezahl und Position — die Komponente kümmert sich um den Rest. Die beiden nächsten Kapitel-Unterseiten (Collapsing Blocks, Match Three) rufen diese Komponente an der Stelle auf, an der Original-Code new PointBurst(this, pointsScored, mouseX, mouseY) schreibt.

Irgendwo in den Kasten klicken — jeder Klick erzeugt eine unabhängige PointBurst-Instanz mit zufälliger Punktezahl.
