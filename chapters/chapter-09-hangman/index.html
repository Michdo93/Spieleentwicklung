Hangman

Der Klassiker: Buchstaben raten, bevor der Galgen fertig gezeichnet ist. Erstaunlich wenig Code für ein komplettes Spiel. Original: A3GPU209_Hangman.

Der ganze Spielzustand ist ein einziger String

Kein Array, kein Grid — nur zwei Strings: phrase (die Lösung) und shown (was der Spieler sieht). Ein Treffer ersetzt an der passenden Stelle den Unterstrich durch den echten Buchstaben. Diese Technik überträgt sich unverändert nach JavaScript.

ActionScript 3 
shown = phrase.replace(
  /[A-Za-z]/g, "_");

// bei Treffer an Position i:
shown = shown.substr(0,i)
  + phrase.substr(i,1)
  + shown.substr(i+1);
JavaScript 
shown = phrase.replace(
  /[A-Za-z]/g, "_");

// bei Treffer an Position i:
shown = shown.substring(0,i)
  + phrase.substring(i,i+1)
  + shown.substring(i+1);

String.replace() mit reguläremAusdruck funktioniert in JavaScript identisch zu AS3 — substr() ist inzwischen offiziell "legacy", substring() macht dasselbe und ist die empfohlene moderne Variante.

Ein Fehlversuch = ein Körperteil

Im Original stand für jeden Fehlversuch ein Frame im Galgenmännchen-Symbol bereit (character.gotoAndStop(numWrong+1)). Auf dem Canvas zeichne ich stattdessen die Körperteile (Kopf, Rumpf, Arme, Beine, Gesicht) als kleine Funktionsliste — bei jedem Fehlversuch kommt eine dazu.

Einfach auf der Tastatur Buchstaben tippen — a–z werden ausgewertet, 8 Fehlversuche sind erlaubt.
