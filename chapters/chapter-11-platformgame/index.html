Platform Game

Das Jump-'n'-Run-Grundgerüst: Schwerkraft, Sprung, Kollision mit Boden/Wänden, ein scrollendes Level, ein Gegner, der an Wänden umkehrt, Schatz/Schlüssel/Tür/Truhe. Original: A3GPU211_PlatformGame.

Die Physik in einer einzigen Funktion

moveCharacter() ist der Kern des ganzen Spiels — und wird für Held und Gegner gleichermaßen benutzt. Die Werte gravity=.004, jumpSpeed=.8, walkSpeed=.15 sind bereits "Pixel pro Millisekunde" gerechnet, genau wie timeDiff selbst — das lässt sich unverändert übernehmen, ohne eine einzige Konstante neu kalibrieren zu müssen.

ActionScript 3 
var verticalChange =
  char.dy*timeDiff
  + timeDiff*gravity;
if (verticalChange > 15.0)
  verticalChange = 15.0;
char.dy += timeDiff*gravity;

if (char.jump) {
  char.jump = false;
  char.dy = -char.jumpSpeed;
  verticalChange = -char.jumpSpeed;
}
JavaScript 
let verticalChange =
  char.dy*timeDiff
  + timeDiff*gravity;
if (verticalChange > 15.0)
  verticalChange = 15.0;
char.dy += timeDiff*gravity;

if (char.jump) {
  char.jump = false;
  char.dy = -char.jumpSpeed;
  verticalChange = -char.jumpSpeed;
}

Zwei Kollisionsschleifen: erst landen, dann anstoßen

Zuerst wird geprüft, ob der Charakter auf einer Plattform landet (vertikale Bewegung gegen alle fixedObjects), danach — mit der bereits korrigierten Y-Position — ob er seitlich gegen eine Wand läuft. Diese Reihenfolge ist wichtig: Sie verhindert, dass man beim Landen gleichzeitig hängenbleibt. 1:1 aus dem Original übernommen.

Kamera folgt dem Helden

scrollWithHero() hält den Helden zwischen zwei unsichtbaren Rändern (100px vom Bildschirmrand) — verlässt er diesen Korridor, verschiebt sich die Kamera, begrenzt auf die Levelgrenzen. Dasselbe Prinzip wie in unzähligen Seitwärts-Scrollern.

Das Level: von Hand nachgebaut

Im Original standen Boden-, Wand-, Schatz- und Türsymbole fertig auf der Flash-Zeitleiste — keine Daten, die sich exportieren lassen. Das Beispiellevel hier (zwei Sprunghindernisse, eine Schatzplattform, ein patrouillierender Gegner, Schlüssel → Tür → Schatztruhe) wurde deshalb neu entworfen, benutzt aber exakt dieselben Objektarten und dieselbe Physik wie das Original.

← / → laufen, Leertaste springt. Auf einen Gegner von oben springen, um ihn zu besiegen — von der Seite berühren kostet ein Leben. Schlüssel vor der Tür einsammeln, Truhe danach öffnet das Spiel als gewonnen.
