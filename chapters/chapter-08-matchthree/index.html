Match Three

Ein Bejeweled-Klon: zwei angrenzende Teile tauschen, um Dreier-Reihen zu bilden. Das komplexeste Spiel in diesem Kapitel — mit Ketten-Reaktionen, garantiert lösbarem Startraster und Rückgängig-Tausch bei Fehlversuchen. Original: A3GPU208_MatchThree.

"Ausprobieren, dann entscheiden"-Muster

makeSwap() ist ein hübscher Trick: statt vorher zu prüfen, ob ein Tausch ein Match ergibt, wird einfach getauscht und danach lookForMatches() aufgerufen. Kein Treffer? Zurücktauschen. Das spart eine komplett separate "Was-wäre-wenn"-Prüffunktion.

ActionScript 3 
function makeSwap(p1,p2) {
  swapPieces(p1,p2);
  if (lookForMatches().length
      == 0) {
    swapPieces(p1,p2); // zurück
  } else {
    isSwapping = true;
  }
}
JavaScript 
makeSwap(p1, p2) {
  this.swapPieces(p1, p2);
  if (this.lookForMatches()
      .length === 0) {
    this.swapPieces(p1, p2);
  } else {
    this.isSwapping = true;
  }
}

Startraster: erst mischen, dann prüfen, ggf. neu mischen

Ein zufälliges 8×8-Raster hätte oft schon von Anfang an Dreier-Reihen (unerwünscht) oder gar keine möglichen Züge (Sackgasse). Das Original löst das simpel: Raster erzeugen, mit lookForMatches() und lookForPossibles() prüfen, bei Bedarf komplett neu würfeln — so lange, bis ein gültiges Startraster gefunden ist.

Mögliche-Züge-Erkennung ohne Ausprobieren

Statt für jeden der 64 Felder alle vier Nachbartausche tatsächlich durchzuspielen, prüft lookForPossibles() geometrische Muster ("zwei gleiche in einer Reihe plus ein potenzieller dritter einen Zug entfernt") — deutlich schneller. Die Musterlisten (relative Koordinaten) sind 1:1 aus dem Original übernommen.

Ketten-Reaktionen fast geschenkt

Weil findAndRemoveMatches() nach dem Nachrutschen erneut sich selbst über den Game-Loop auslöst (isDropping-Flag), entstehen Kettenreaktionen automatisch — ohne eigene Rekursionslogik dafür schreiben zu müssen.

Ein Teil anklicken, dann ein direkt angrenzendes zweites — ergibt der Tausch eine Dreier-Reihe, bleibt er bestehen, sonst tauscht das Spiel automatisch zurück.
