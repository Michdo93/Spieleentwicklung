Trivia Game Deluxe

Dasselbe Quiz-Grundgerüst wie Chapter 10b, jetzt mit Zeitdruck: 25 Sekunden pro Frage, Punkte sinken mit der Zeit, ein Hinweis-Button kostet Punkte, ein Faktum nach jeder Antwort. Original: A3GPU210_TriviaGameDeluxe.
⚠️ Braucht wie Chapter 10a einen lokalen Server.

10 aus 42: zufällige Teilmenge ziehen

Der Fragenpool (trivia2.json, aus trivia2.xml konvertiert) enthält 42 Fragen — jede Runde werden per Zufall nur 10 davon ausgewählt. Derselbe "ziehen und aus dem Pool entfernen"-Trick wie beim Antworten-Mischen, nur auf Fragen statt Antworten angewendet.

ActionScript 3 
var chosenXML = <trivia></trivia>;
while (chosenXML.child("*")
    .length() < numToChoose) {
  var r = Math.floor(
    Math.random()*allXML
    .child("*").length());
  chosenXML.appendChild(
    allXML.item[r].copy());
  delete allXML.item[r];
}
JavaScript 
function selectQuestions(
    all, numToChoose) {
  const copy = [...all];
  const chosen = [];
  while (chosen.length < numToChoose
      && copy.length > 0) {
    const r = Math.floor(
      Math.random()*copy.length);
    chosen.push(copy[r]);
    copy.splice(r, 1);
  }
  return chosen;
}

Countdown mit sinkenden Punkten

25 Sekunden Zeit, jede Sekunde kostet 25 Punkte vom möglichen Maximum (1000). Das Original nutzt einen Timer(1000,25) mit einer Clock-Bibliotheksanimation; hier übernimmt ein setInterval(fn, 1000) die Zählung, ein CSS- width-Übergang die visuelle Fortschrittsleiste.

Hint-Button: Punkte gegen Information tauschen

Ein Hinweis kostet 300 Punkte sofort — eine kleine Risikoabwägung für den Spieler. Genau wie im Original bleibt der Button nur verfügbar, solange noch nicht geantwortet wurde.

Jede Runde 10 zufällige von 42 Fragen, 25 Sekunden Zeit, Hinweis kostet Punkte, nach jeder Antwort ein kurzes Faktum zum Thema.
