Air Raid 2

Dieselbe Flak-Kanone wie in Kapitel 05 — aber jetzt dreht sie sich, statt seitlich zu fahren, und Kugeln fliegen in die tatsächliche Feuerrichtung. Original: A3GPU207_AirRaid2.

Eine kleine Änderung, dieselbe Struktur

Das Original ändert an AirRaid.as praktisch nichts außer der Kanonensteuerung — AAGun bewegt jetzt rotation statt x, und Bullet bekommt einen Rotationswinkel statt einer festen "immer nach oben"- Geschwindigkeit. Die komplette Spiellogik drumherum (Flugzeug-Spawning, Kollision, Punktestand) ist unverändert aus Kapitel 05 übernommen.

AAGun · Chapter 05 (Air Raid) 
if (leftArrow) {
  newx -= speed*dt;
}
this.x = newx;
AAGun · Chapter 07 (Air Raid 2) 
if (leftArrow) {
  newRotation -= rotSpeed*dt;
}
this.rotation = newRotation;

Kugeln fliegen jetzt "schräg"

Das ist die direkte Anwendung von Chapter 07a: Die Kugel startet nicht an der Kanonenspitze selbst, sondern einen festen Abstand (initialMove = 35) in Feuerrichtung entfernt — sonst würde sie mitten im Kanonenrohr erscheinen. Geschwindigkeit in X und Y ergibt sich direkt aus cos/sin des Rotationswinkels.

ActionScript 3 
var rad:Number =
  2*Math.PI*rot/360;
this.x = x + 35*Math.cos(rad);
this.y = y + 35*Math.sin(rad);
dx = speed*Math.cos(rad);
dy = speed*Math.sin(rad);
JavaScript 
const rad = deg2rad(rotDeg);
this.x = x + 35*Math.cos(rad);
this.y = y + 35*Math.sin(rad);
this.dx = speed*Math.cos(rad);
this.dy = speed*Math.sin(rad);
Steuerung wie gehabt: ← / → drehen die Kanone (begrenzt auf −170° bis −20°, sie kann also nicht nach unten zielen), Leertaste feuert.
