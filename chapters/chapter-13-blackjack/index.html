Blackjack

Vollständiges Blackjack: 6-Deck-Schuh, Einsätze, verdeckte Dealer-Karte, Hit/Stay, Dealer-KI (zieht bis 17), Ass zählt 1 oder 11. Original: A3GPU213_Blackjack.

Karten als kurze Strings — bleibt exakt gleich

Das Original kodiert jede Karte als String wie "h1" (Herz Ass) oder "s13" (Pik König) — Farbe als Buchstabe, Wert als Zahl. Dieses Format wurde unverändert übernommen, nur das Zeichnen der Karte ist anders: statt card.gotoAndStop(cardVal) gegen ein 52-Frame- Bibliothekssymbol wird ein HTML-Element mit Rang und Symbol gerendert.

ActionScript 3 
var suits = ["c","d","s","h"];
var temp = [];
for (var i=0; i<6; i++)
  for (var suit=0; suit<4; suit++)
    for (var num=1; num<14; num++)
      temp.push(suits[suit]+num);
JavaScript 
const suits = ["c","d","s","h"];
const temp = [];
for (let i=0; i<6; i++)
  for (const suit of suits)
    for (let num=1; num<14; num++)
      temp.push(suit+num);

Die Timed-Event-Queue: dasselbe Muster wie Timer

Statt jede Karte sofort auf einmal auszuteilen, reiht das Original Ereignisse wie "deal card to dealer" in eine Liste und arbeitet sie im Sekundentakt per Timer ab — das erzeugt das typische "eine Karte nach der anderen"-Gefühl am Spieltisch. Dasselbe Muster funktioniert mit setInterval() identisch.

ActionScript 3 
timedEvents = new Timer(1000);
timedEvents.addEventListener(
  TimerEvent.TIMER,
  playTimedEvents);
timedEvents.start();

function playTimedEvents(e) {
  var thisEvent =
    timedEventsList.shift();
  if (thisEvent ==
      "deal card to dealer") {
    dealCard("dealer");
  } // ...
}
JavaScript 
this.timedEvents = setInterval(
  () => this.playTimedEvents(),
  700
);

playTimedEvents() {
  const thisEvent =
    this.timedEventsList.shift();
  if (thisEvent ===
      "deal card to dealer") {
    this.dealCard("dealer");
  } // ...
}

Ass-Logik und Dealer-KI: unverändert

handValue() zählt ein Ass zunächst als 1, erhöht die Summe aber um 10 (macht das Ass effektiv zu 11), falls das nicht zum Überkaufen führt — die klassische Blackjack-Ass-Regel, 1:1 übernommen. Die Dealer-KI zieht stur, solange ihr Handwert unter 17 liegt.
