Text Examples

Zwei Demos: ein direkter Vergleich der AS3-String-Methoden mit ihren JS-Äquivalenten, und eine kleine Buchstaben-Flug-Animation für Titelbildschirme. Original: A3GPU209_TextExamples (StringExamples, TextFly).

String-Methoden: fast alles überträgt sich 1:1

Der große Vorteil beim Portieren dieses Beispiels: AS3s String-API wurde stark von JavaScript beeinflusst — charAt, indexOf, lastIndexOf, slice, split, join, replace mit regulären Ausdrücken funktionieren identisch. Nur zwei Kleinigkeiten unterscheiden sich:

  • substr(start, length) ist in JS als "legacy" markiert — funktioniert noch, aber substring(start, end) ist die empfohlene Variante
  • AS3s substring() tauscht Start/Ende automatisch, wenn start > end — JS' substring() macht das ebenfalls automatisch, arbeitet intern also schon gleich
ActionScript 3 
trace(myString.search(/r.*n/));
trace(myString.replace(
  /(raven)(.*)(writing desk)/g,
  "$3$2$1"
));
JavaScript 
console.log(myString.search(/r.*n/));
console.log(myString.replace(
  /(raven)(.*)(writing desk)/g,
  "$3$2$1"
));

TextFly: reine Interpolation, kein Physik-Trick

Jeder Buchstabe bekommt Start- und Endposition, Start- und Endskalierung, eine Gesamtrotation — und bei jedem Animationsschritt wird linear zwischen beiden Werten interpoliert (start*(1-p) + end*p). Dieselbe Interpolationsformel wie beim PointBurst-Effekt aus Kapitel 08, nur auf mehrere Eigenschaften gleichzeitig angewendet.
