Collapsing Blocks

Das "SameGame"-Prinzip: verbundene Blöcke gleicher Farbe anklicken, sie verschwinden, der Rest rutscht nach. Original: A3GPU208_CollapsingBlocks.

Flood-Fill: eine der nützlichsten Spiele-Techniken überhaupt

testBlock() ist eine rekursive Flood-Fill-Suche: vom angeklickten Block aus in alle vier Richtungen weiterspringen, solange der Nachbar denselben Typ hat. Dieselbe Technik steckt hinter dem Fülleimer-Werkzeug in Bildbearbeitungsprogrammen, hinter Minesweepers "leere Fläche aufdecken" und vielen Rätselspielen.

ActionScript 3 
function testBlock(col,row,type) {
  var testList = [];
  if (getBlockType(col,row)
      == 0) return testList;
  if (blocks[col][row].type
      == type) {
    testList.push(
      blocks[col][row]);
    blocks[col][row].type = 0; // als besucht markieren
    testList = testList.concat(
      testBlock(col+1,row,type));
    testList = testList.concat(
      testBlock(col-1,row,type));
    // ... row+1, row-1
  }
  return testList;
}
JavaScript 
testBlock(col, row, type, visited) {
  const key = col+","+row;
  if (visited.has(key)) return [];
  if (this.getBlockType(col,row)
      !== type) return [];
  visited.add(key); // als besucht markieren
  let list = [this.blocks[col][row]];
  list = list.concat(
    this.testBlock(col+1,row,type,visited));
  list = list.concat(
    this.testBlock(col-1,row,type,visited));
  // ... row+1, row-1
  return list;
}

Kleine, aber wichtige Änderung: Das Original markiert besuchte Blöcke, indem es kurzzeitig type = 0 setzt und bei einem Fehlversuch (weniger als zwei Treffer) wieder zurücksetzt. Ich benutze stattdessen ein separates visited-Set pro Klick — funktional gleichwertig, aber ohne Nebenwirkung, falls beim Zurücksetzen mal ein Fall übersehen würde.

Nachrutschen in zwei Phasen

Erst fallen die Blöcke einer Spalte nach unten (affectAbove() aktualisiert nur die Zieldaten, die eigentliche Bewegung passiert im Game-Loop), dann — sobald alles zur Ruhe gekommen ist — rutschen leere Spalten von links zusammen (checkForEmptyColumns()). Diese zwei-Phasen-Reihenfolge ist 1:1 aus dem Original übernommen.

Zwei oder mehr verbundene, gleichfarbige Blöcke anklicken, damit sie verschwinden. Punkte = Anzahl Blöcke im Quadrat.

Verbundene Blöcke gleicher Farbe anklicken.
