Sliding Puzzle

Das klassische 15-Puzzle-Prinzip (hier 4×3 = 11 Teile + ein Leerfeld): Teile neben dem Leerfeld anklicken, um sie mit einer kurzen Gleitanimation hineinzuschieben. Original: A3GPU206_SlidingPuzzle.

Mischen ohne sichtbare Animation

Ein cleverer Kniff im Original: Statt die Teile zufällig zu platzieren (was zu unlösbaren Zuständen führen könnte), macht shufflePuzzlePieces() 200 gültige Züge in Folge — jeder Zug ist eine normale Verschiebung, nur ohne die Slide-Animation. Das garantiert automatisch, dass das Puzzle lösbar bleibt, weil nur erlaubte Bewegungen benutzt werden.

Gleitanimation: fester Timer → Zeit-Fortschritt

Für die Animation selbst nutzt AS3 einen Timer mit exakt slideSteps (10) Ticks über slideTime (250ms) — bei jedem Tick wandert das Teil um 1/10 der Strecke weiter. Im Web ist das eleganter mit requestAnimationFrame und einer Fortschritts-Berechnung aus der verstrichenen Zeit lösbar (funktioniert auch bei schwankender Bildrate sauber, nicht nur bei genau 10 Ticks).

ActionScript 3 
slideAnimation = new Timer(
  slideTime/slideSteps,
  slideSteps);
slideAnimation.addEventListener(
  TimerEvent.TIMER, slidePiece);

function slidePiece(e) {
  piece.x += dx/slideSteps;
  piece.y += dy/slideSteps;
}
JavaScript 
const t = Math.min(
  (now - start) / slideTime, 1
);
piece.animX =
  fromX + (toX-fromX) * t;
piece.animY =
  fromY + (toY-fromY) * t;
