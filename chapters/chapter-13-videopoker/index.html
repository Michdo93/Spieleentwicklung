Video Poker

Klassisches 5-Karten-Draw-Poker mit vollständiger Handbewertung — von "Nothing" bis Royal Flush. Der komplexeste Analyse-Code in diesem Kapitel. Original: A3GPU213_VideoPoker.

Handbewertung: reine Datenverarbeitung, 1:1 übertragbar

handValue() ist der wertvollste Teil dieses Kapitels — eine komplette Poker-Handanalyse (Straight, Flush, Paare, Drillinge, Vierlinge, Royal Flush) in überschaubarem Code. Weil das reine Array-/Zahlen-Verarbeitung ist, ohne jeden Bezug zu Flash- spezifischen APIs, ließ sich diese Funktion nahezu unverändert übernehmen.

ActionScript 3 
var counts:Array = [];
for(i=0;i<14;i++) counts.push(0);
for(i=0;i<5;i++) counts[nums[i]]++;

var pair=false, twoPair=false;
for(i=1;i<14;i++) {
  if (counts[i] == 2) {
    if (pair) twoPair = true;
    else pair = true;
  }
  // ...
}
JavaScript 
const counts = new Array(14).fill(0);
nums.forEach(n => counts[n]++);

let pair = false, twoPair = false;
for (let i=1; i<14; i++) {
  if (counts[i] === 2) {
    if (pair) twoPair = true;
    else pair = true;
  }
  // ...
}

Karten zum Abwerfen markieren

Im Original wird eine angeklickte Karte "umgedreht" (gotoAndStop(2)), um sie zum Abwerfen zu markieren — hier stattdessen eine halbtransparente "DISCARD"-Markierung per CSS. Funktional identisch: Karte anklicken markiert sie, nochmal anklicken hebt die Markierung wieder auf.

Sortieren für die Analyse — nicht für die Anzeige

compareHands() sortiert eine Kopie der Hand nach Zahlenwert, bevor handValue() nach Straights und Paaren sucht — die tatsächlich angezeigten Karten bleiben in der Reihenfolge, in der sie ausgeteilt wurden. Ein kleines, aber wichtiges Detail, das 1:1 übernommen wurde.
