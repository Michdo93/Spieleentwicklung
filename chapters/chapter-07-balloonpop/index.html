Balloon Pop

Eine Kanone abfeuern, deren Kugel jetzt der Schwerkraft folgt statt geradeaus zu fliegen — eine parabelförmige Flugbahn, drei Level mit wachsender Ballonzahl. Original: A3GPU207_BalloonPop.

Von gerader Linie zu Parabelflug

Der einzige, aber entscheidende Unterschied zu Air Raid 2: Die Kugel-Geschwindigkeit in Y-Richtung bleibt nicht konstant, sondern wächst jeden Frame um einen festen Betrag — Schwerkraft. Startrichtung kommt weiterhin aus cos/sin des Kanonenwinkels (Chapter 07a), aber danach übernimmt die Physik.

ActionScript 3 
cannonballDX = speed*Math.cos(
  2*Math.PI*cannon.rotation/360);
cannonballDY = speed*Math.sin(
  2*Math.PI*cannon.rotation/360);

// jeden Frame:
cannonball.x += cannonballDX;
cannonball.y += cannonballDY;
cannonballDY += gravity;
JavaScript 
const rad = deg2rad(rotation);
ball.dx = speed*Math.cos(rad);
ball.dy = speed*Math.sin(rad);

// jeden Frame (zeitbasiert):
ball.x += ball.dx * dt;
ball.y += ball.dy * dt;
ball.dy += GRAVITY * dt;

Wichtig: Das Original erhöht cannonballDY pro Frame um einen festen Wert (.05) — das ist implizit an 30 fps gebunden. Ich rechne stattdessen mit GRAVITY * dt in echten Pixel/Sekunde², damit die Flugbahn bei jeder Bildrate gleich aussieht.

Ballon-Layout: selbst generiert

Im Original standen die Ballons als fertig platzierte Objekte auf der Zeitleiste jedes Levels — von Hand im Flash-Editor positioniert. Ohne diesen visuellen Editor generiere ich für jedes der drei Level ein eigenes Koordinaten-Layout in JavaScript; Levelanzahl, Farben-Zufall und "wachsende Ballonzahl pro Level" entsprechen dem Original.
