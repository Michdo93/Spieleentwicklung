Matching Game

Das klassische Memory-Kartenspiel: 36 Karten, 18 Paare, aufdecken und merken. Das Buch baut dieses Spiel im Original in zehn Schritten auf (MatchingGame1 … MatchingGame10) — wir portieren direkt die fertige Fassung MatchingGameObject10 + Card10.

Warum "Version 10" statt "Version 1"?

Das Buch nutzt hier bewusst eine Schritt-für-Schritt-Methode: Version 1 zeigt nur das Grundraster, Version 2 fügt Klick-Erkennung hinzu, Version 5 die Flip-Animation, Version 8 Sound, Version 10 Highscore und Timer. Didaktisch wertvoll beim Lesen des Buchs — für die Portierung reicht es, die letzte, vollständige Version 1:1 zu übertragen; die Lernschritte stehen weiter unten als Kurzfassung.

Die Flip-Animation: scaleX statt echtem 3D

Card10.as simuliert eine Kartendrehung, indem es scaleX in zehn Schritten von 1 auf 0 und wieder auf 1 fährt — in der Mitte (bei scaleX = 0) wechselt die sichtbare Kartenseite. Genau dasselbe Prinzip funktioniert im Canvas mit ctx.scale(), nur zeitbasiert statt frame-gezählt.

ActionScript 3 
function flip(e:Event) {
  flipStep--;
  if (flipStep > 5) {
    scaleX = .2*(flipStep-6);
  } else {
    scaleX = .2*(5-flipStep);
  }
  if (flipStep == 5) {
    gotoAndStop(flipToFrame);
  }
}

update(dt) {

Kartenbilder: eigene Symbole statt Bibliothek

Im Original waren die 18 Kartenmotive einzelne Frames im Card10-Bibliothekssymbol (von Hand gezeichnete Bilder). Ohne Flash-Bibliothek generiere ich stattdessen 18 eindeutige Symbol/Farb-Kombinationen (6 Formen × 3 Farben) direkt mit der Canvas-API — funktional identisch: 18 klar unterscheidbare Motive.

Spiellogik: derselbe Zustandsautomat

Der Kern von clickCard() ist eine kleine Zustandsmaschine — "keine Karte gewählt" → "eine Karte gewählt" → "zwei Karten, vergleichen" — die 1:1 übernommen wurde, inklusive des Sonderfalls, dieselbe Karte zweimal anzuklicken (schließt sie wieder).

Sounds sind die Original-Audiodateien aus dem Buch (FirstCardSound, MissSound, MatchSound), von AIFF nach MP3 konvertiert — AIFF spielt in den meisten Browsern nicht zuverlässig ab.
